# Task 1: Data Visualization Objective

To create a bar chart and histogram using a dataset.



## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook



## Dataset

Population dataset loaded from an online source.



## Process

* Loaded dataset using pandas
* Filtered data for a specific year
* Created bar chart for top 10 countries
* Created histogram for population distribution



## Output

* Bar chart showing top countries by population
* Histogram showing distribution of population
